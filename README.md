## Starten der Umgebung
Das Notebook kann per mybinder (https://mybinder.org/) geöffnet werden.
Folgende Daten müssen in die Felder eingetragen werden:
* Github Repo: https://github.com/simon1805/Logistic-Regression.git
* Git ref: main
* File to open: index.ipynb 
## Dokumentation
- Ziel dieses Modells ist es, einem Unternehmen helfen zu entscheiden, ob sie ihren Fokus auf die App oder Website legen sollen
- Dazu folgt zuerst eine "Explorative Daten Analyse", um eine Korrelation zwischen den Daten zu finden
- Danach werden die Daten in Test- und Trainingsdaten gesplittet
- Mit den Trainingsdaten wird das Lineare Regressionsmodell trainiert und mit den Test Daten getestet
- Am Ende wird das Modell bewertet
## Erwartetes Ergebnis

- Die explorative Datenanalyse zeigt erste Zusammenhänge zwischen der Nutzung der App und der Website sowie deren Einfluss auf das Nutzerverhalten.
- Das lineare Regressionsmodell liefert eine Vorhersage darüber, ob das Unternehmen seinen Fokus eher auf die App oder die Website legen sollte.
- Es werden typische Ausgaben eines Regressionsmodells erzeugt, darunter:
  - Regressionskoeffizienten
  - Intercept
  - Modellgüte (z. B. R²-Wert)
  - Residuenanalyse bzw. Fehlermaße
- Die Ergebnisse sind reproduzierbar, wenn alle Zellen des Notebooks vollständig ausgeführt werden.
